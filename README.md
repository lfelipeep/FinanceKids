# codigos para ejecutar y/o ver la pagina y lo que se le ajusta
## pasos
### crear el virtualenv
```python3 -m venv .venv```
### activarlo (Bash)
```source .venv/bin/activate```
### Instalar Django con pip
```python -m pip install django```
### Instala el paquete
```pip install python-dotenv```
### correr el servidor:
```python manage.py runserver 0.0.0.0:8000```
# si todo lo hiciste bien asi podras visulizar la pagina
## enlace para acceder como admin
http://127.0.0.1:8000/admin/
